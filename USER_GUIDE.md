## Getting in

1. Open the URL on your phone or computer
2. Enter your **username** (all lowercase — `adam`, `alivia`, or `chris`)
3. Enter your **password** (Adam sets this for you)
4. You'll land on your personal **Home** tab

### Install to your home screen (recommended)

On **iPhone Safari**: tap the Share button → **Add to Home Screen** → Add.
On **Android Chrome**: tap the menu (⋮) → **Install app** → Install.

Once installed, opening The Floor from the home screen looks like a real app — no browser bar, notifications work better, and the app remembers you.

## Home tab

## Tasks tab

### Create a task
1. Tap the big gold **+** button (FAB) at the bottom-right, or **Quick Log → Task**
2. Type a title
3. (Optional) Pick one or more people to assign — tap pills to toggle multiple
4. (Optional) Set priority: Normal / Hot / Routine
5. (Optional) Set **Scheduled** date/time (when to do it) and **Due** date/time (must be done by)
6. Hit **Save**

> 💡 **Tap anywhere in the date field** to open the calendar picker — don't hunt for the tiny icon.

### Complete a task
Two ways:
- **Tap the circle** on the left → checkmark appears instantly with a little pop animation
- **Swipe right** across the row → the row slides off with a green "Complete" track

### Edit or delete
Tap **the body** of the task (not the circle) → edit modal opens with all fields. Delete button is at the bottom.

### Filters + search
At the top of the Tasks tab:
- **All / Mine / Today / This Week / Overdue / Hot / per-person / Open / Done** — tap a chip
- Search bar filters by title or notes (case-insensitive)

### Bulk clear
At the bottom of the Tasks tab: **Clear Completed** deletes all your done tasks (Adam can clear everyone's).

## Calendar tab

### Month view (default)
- Prev / next arrows navigate months
- **Today** button jumps back to the current week
- Days with events/tasks show gold dots (up to 3)
- **🦋 Gold tint** = Kaleesi day (Adam)
- **Muted tint** = Marta's day
- **Split gradient** = exchange day
- **Gold outline** = today
- **Solid gold fill** = selected day

Tap any day → detail panel appears below with:
- Kaleesi status badge
- All events (gold)
- All tasks due (amber ⚑ or green ✓ if done) — tap a task to edit
- **Add Event for This Day** button (pre-fills the date)

### Week view
Scrolling 7-day list. Everything in the next week in one place.

### Adding events
Events are things like "Kaleesi pediatrician at 2pm" or "Lawn service Friday". Pick **Event** from Quick Log or the + button.

### Editing events
Tap any event in the day-detail panel → edit modal opens (Adam, or event creator only). Delete button available.

## Vault tab

Where **household reference info** lives: wifi password, alarm codes, vehicle info, vendor contacts, etc.

**Section chips** at the top: House / Fleet / Supplies / Vendors / Logins / Kid.

### Visibility
Each vault entry has a **visible to** list. Adam sees everything. Alivia and Chris only see what Adam has shared with them. Example: Alivia's Vivint PIN is visible to Adam + Alivia only; Chris's is visible to Adam + Chris only.

### Editing (Adam only)
Tap any entry → edit modal opens with:
- Section (where it belongs)
- Title and content
- Visible To pills (tap to toggle who can see this)
- Delete button

At the bottom of each section: **+ Add Vault Entry** dashed button.

### Shopping list
Below the vault entries on the Vault tab. Anyone can check off items they've bought.

## Wire tab (Group chat + Notes + Ideas)

Three sub-sections toggle at the top:

### Chat
- Type a message → Enter to send, or tap the gold send button
- **@mention** someone: type `@adam`, `@alivia`, or `@chris` — they get a notification
- Tap the **quick-pill buttons** above the input to auto-insert a mention
- **Attach photo** with the camera button → photo uploads + appears inline in the bubble
- **Delete your own message**: tap the trash icon next to it (Adam can delete any message)
- Consecutive messages from the same person group together (avatar shows once)

### Notes
Persistent notes — not transient chat. Use for things like:
- "Reminder for Alivia: new formula brand"
- "Leak under sink, plumber coming Friday"

### Ideas
Brainstorm space. Capture ideas as they come with the 💡 Idea quick-log option.

## Ask tab (Requests)

For submitting things that need Adam's approval, time off, or reminders.

### Filter chips
- **Pending** (default) — what's waiting for action
- **Approved / Denied / All** — history

### Submit a request
Pick **Ask** from Quick Log or the + button.
- Title + notes (what you're asking)
- Amount ($) — if it's a purchase approval
- **Attach a photo** — e.g. snap the item you want to buy (exactly like "Chris wants to buy this couch for $800")
- Save

### Edit your own pending request
Tap the pencil icon on the top-right of your own pending request → edit any field, save, or delete.

### Adam's view
Approve or Deny buttons appear for him. Staff see the current status instead.

## Shift tab (Clock in/out)

### Clock in
1. Go to Shifts tab
2. Tap **Clock In**
3. Timer starts immediately (the app captures your GPS in the background)
4. Badge updates to **At home ✓** (within 150m of 4826 Weaver St), **Off-site ⚠**, or **Location unavailable**

### Clock out
Tap **Out** — toast confirms "Clocked out ✓" and your shift saves with the duration auto-computed.

> ⚠️ **Clock in/out is manual.** The app doesn't track you automatically. If you forget to clock out, tell Adam.

### Long shift warning
If you're clocked in for more than 10 hours, a red banner appears asking if you forgot to clock out.

### Weekly summary
Above the Recent Shifts list: **This Week** hours + shift count, **Last Week** hours. Sunday-reset.

### Principal reminder (Adam)
Adam has a **Send Clock-In Reminder to Team** button that broadcasts a reminder to the Wire chat with one tap.

## Admin tab (Adam only)

### Integrations
- **Kaleesi Calendar** — auto-connected (no config needed). Reads custody schedule from the Kaleesi app.
- **Install App** — opens install instructions for iPhone / Android.

List of all household members with status pills. Tap any row to edit:

### Roles
Built-in roles: Principal, Nanny, PA, Groundskeeper. Each has an icon and description.

### Recent Activity (audit log)
Last 15 actions across the app — who did what, when. Auto-updated via realtime.

### Changing your own password
Adam has a 🔑 key icon in the header (top-right next to Sign Out). Tap it → enter current + new password → Save.

## Passwords & security

- Each user has their own username and password
- Passwords are **SHA-256 hashed** with a per-user salt (never stored in plain text)
- Sessions persist across reloads — you stay signed in until you sign out
- Sign out from the header button (top-right, door icon) — confirms before signing out
- Only Adam can reset others' passwords

## Troubleshooting

### Clock-in button doesn't do anything
Hard refresh: on iPhone pull down on the page with two fingers, on desktop Ctrl+Shift+R. Sign out and back in.

### I'm still showing as clocked in from yesterday
Adam can see all active shifts in the Home briefing ("On the clock: Adam · 18h ⏰"). Ask him to force-close it. A future version will let you do this yourself.

### Photos won't upload
They must be under 8 MB. Try a smaller resolution or re-take.

### Notifications aren't coming through
1. Make sure you allowed notifications when the browser prompted
2. On iPhone, notifications only work when the app is installed to the home screen (PWA), not in a Safari tab
3. Sign out + back in to re-trigger the permission prompt

### I can't see what everyone else can see
That's the **visibility system** working correctly. Each Vault entry has a visible-to list; some things are intentionally hidden from you for security (e.g. other people's personal PIN codes).
